rubysmb by canweriotnow

Ruby SMB client library

Yeah... SMB sucks, but in the "real world," people insist on Windows shares. Let's make it happen. Ruby/SMB hasn't been maintained since 2007, but I need it to make life easier at work, so I'm trying to update this sucker.

Dependencies

libsmbclient libsmbclient development headers Ruby >= 1.9.1

Install

ruby extconf.rb make (sudo) make install (I'm working on Gemifying it.
